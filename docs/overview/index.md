# Installation du Cloud Privé

## Prérequis

- Serveurs physiques ou virtuels.
- Systèmes d'exploitation supportés : **CentOS**, **Ubuntu**.
- Accès administrateur.

## Étapes d'installation

1. **Télécharger l'ISO** :

   wget http://example.com/cloud-private.iso
