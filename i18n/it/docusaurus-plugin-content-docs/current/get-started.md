# Iniziare con Hikube

## Benvenuti su Hikube

Hikube offre una soluzione cloud moderna che facilita il deployment e l'orchestrazione di applicazioni containerizzate e macchine virtuali.

## Versioni della documentazione

Questa documentazione è ora versionata! Puoi consultare:
- **Versione 1.0**: Documentazione semplificata
- **Versione attuale (Versione 2.0.2)**: Documentazione in sviluppo con le ultime funzionalità

## Funzionalità principali

### 🚀 **Deployment semplificato**
- API Kubernetes nativa e compatibile
- Deployment tramite `kubectl apply`
- Infrastructure as Code (IaC) con YAML

### 📦 **Servizi**
- **Database**: PostgreSQL, MySQL, Redis, ClickHouse
- **Servizi di messaggistica**: Kafka, RabbitMQ, NATS
- **Archiviazione**: Cifrata, replicata su 3 datacenter
- **Archiviazione S3**: Bucket S3 compatibili
- **Reti**: Load balancer TCP, cache HTTP, VPN
- **Risorse di calcolo**: Macchine virtuali, GPU

### 🔧 **Infrastructure as Code**
- Gestione dichiarativa delle risorse
- Deployment riproducibile

### 🏗️ **Architettura multi-tenant**
- Isolamento completo tra i tenant
- Possibilità di creazione di sotto-tenant
- Gestione delle quote e delle risorse
- Sicurezza rafforzata
