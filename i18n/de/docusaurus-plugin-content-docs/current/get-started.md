# Erste Schritte mit Hikube

## Willkommen bei Hikube

Hikube bietet eine moderne Cloud-Lösung, die das Deployment und die Orchestrierung von containerisierten Anwendungen sowie virtuellen Maschinen vereinfacht.

## Dokumentationsversionen

Diese Dokumentation ist jetzt versioniert! Sie können einsehen:
- **Version 1.0**: Vereinfachte Dokumentation
- **Aktuelle Version (Version 2.0.2)**: Dokumentation in Entwicklung mit den neuesten Funktionen

## Hauptfunktionen

### 🚀 **Vereinfachtes Deployment**
- Native und kompatible Kubernetes-API
- Deployment über `kubectl apply`
- Infrastructure as Code (IaC) mit YAML

### 📦 **Services**
- **Datenbanken**: PostgreSQL, MySQL, Redis, ClickHouse
- **Messaging-Services**: Kafka, RabbitMQ, NATS
- **Speicher**: Verschlüsselt, repliziert über 3 Rechenzentren
- **S3-Speicher**: S3-kompatible Buckets
- **Netzwerke**: TCP Load Balancer, HTTP-Caches, VPN
- **Compute-Ressourcen**: Virtuelle Maschinen, GPU

### 🔧 **Infrastructure as Code**
- Deklarative Ressourcenverwaltung
- Reproduzierbares Deployment

### 🏗️ **Multi-Tenant-Architektur**
- Vollständige Isolation zwischen Tenants
- Erstellung von Sub-Tenants möglich
- Verwaltung von Quoten und Ressourcen
- Erhöhte Sicherheit
